# Agent Instructions for dargstack

dargstack is a Docker Swarm stack helper CLI written in Go. It simplifies development and production deployments using Docker Compose YAML files merged with [geofffranks/spruce](https://github.com/geofffranks/spruce).

## Documentation Map

**For learning dargstack concepts and project structure:**
- [README.md](README.md) — Project philosophy, quick start, service file format, configuration, profiles, secret templating, environment files, and all CLI commands

**For contributing and development:**
- [CONTRIBUTING.md](CONTRIBUTING.md) — Vision & design principles, architecture, key design decisions, implementation guides, code style, testing, and pull request workflow
